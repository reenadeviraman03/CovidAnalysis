# Covid Analysis Project

## 📌 Project Overview
This project analyzes COVID-19 country-wise data to understand confirmed cases, deaths, and recovery trends.

## 📊 Dataset
- File Name: Day 68 country_wise_latest (1).csv
- Type: CSV
- Data contains country-wise COVID statistics.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## 📈 Key Analysis
- Total confirmed cases by country
- Death rate analysis
- Recovery rate comparison
- Top 10 affected countries visualization

## 🎯 Conclusion
This analysis helps to understand the impact of COVID-19 globally and compare country-wise performance.
